1. Problem Statement Canvas

고객

지역마다 다른 분리수거 규칙을 이해하기 어렵고, 특정 물품의 처리 방법을 잘 모르는 사람들

컨텍스트

버려야 할 물품이 생기거나, 주거지를 옮기면서 해당 지역(시·구)의 분리수거 제도를 확인해야 할 때

근본적인 문제

물품 하나하나의 분리수거 방법을 직접 검색하며 확인해야 하는 불편함

미치는 영향

번거롭고 답답하여 분리수거를 소홀히 하게 되고, 이는 환경 문제로 이어질 수 있음

기존 대안

시·구청 홈페이지나 안내 자료에서 분리수거 제도를 확인

대안의 한계

필요한 순간마다 직접 찾아야 하고, 정보가 흩어져 있어 접근성이 떨어짐

2. 기능

  • 지역별 다른 분리배출 요일·방법 안내 (구/시 단위 차이를 반영)
  • 물품을 선택하면 카드뉴스 형태로 직관적인 분리수거 방법 제공
  • 많이 검색된 물품을 상단에 노출하여 빠르게 확인 가능
  • AI 챗봇을 통해 24시간 실시간 질의응답 가능
  • 자주 검색하는 물품은 즐겨찾기(저장) 기능 제공 → 로그인 필요

3. 솔루션

  • 실시간 위치 정보(또는 지역 선택)를 기반으로 해당 지역의 분리수거 제도를 간단하게 안내
  • 특정 물품명을 검색하면 정확한 분리수거 방법 제공
  • 검색 결과가 없을 경우, AI 챗봇이 즉시 답변 제공

4. UX 솔루션

  • 실시간 위치 확인 → 해당 지역의 분리수거 제도를 카드뉴스 형태로 제공
  • 특정 물품 검색 또는 카테고리 탐색 → 카드뉴스로 직관적 안내
  • 검색창 하단에 ‘최근 많이 검색된 물품’ 연결 → 빠른 접근성 제공
  • 사이드바에 AI 상담 메뉴 제공 → 실시간으로 물어보고 답변 확인 가능
